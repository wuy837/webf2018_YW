The Egg is a chair designed by Arne Jacobsen in 1958 for the Radisson SAS hotelin Copenhagen, Denmark. It is manufactured by Republic of Fritz Hansen.The Egg was designed in a typical Jacobsen style, using state-of-the-art material.It is believed to be inspired by Eero Saarinen's "Womb chair", from which it borrows some traits.
